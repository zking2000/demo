# OpenTelemetry Operator GitHub Pages Demo

这是一个单页静态站点，用通俗易懂的方式介绍 **OpenTelemetry Operator**，适合直接作为 GitHub Pages 首页。

## 内容特色

- 中文通俗介绍
- 深色科技风视觉设计
- 自带 3 张 SVG 精美配图
- 无需构建，纯静态 HTML

## 文件结构

- `index.html`：主页
- `assets/hero-telemetry.svg`：头图
- `assets/before-after.svg`：前后对比图
- `assets/operator-flow.svg`：工作流图

## GitHub Pages 使用方式

1. 推送到 GitHub 仓库
2. 在仓库 Settings → Pages 中启用 Pages
3. 选择：
   - **Deploy from a branch**
   - Branch: `main`
   - Folder: `/ (root)`

启用后首页会自动使用 `index.html`。
