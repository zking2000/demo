OpenTelemetry Operator
Position

It sits in the middle.
It controls onboarding.

Loop

Observe. Create. Mutate. Correct.

Components

At minimum, these are the 4 key parts.

Controller Manager

Watches CRDs and cluster state, then triggers reconcile. This is the main control core of the Operator.

Admission Webhook

Mutates objects during Pod creation and handles auto-instrumentation, sidecar injection, and validation.

OpenTelemetryCollector CRD

Declares how the Collector should run: as a Deployment, DaemonSet, StatefulSet, or Sidecar.

Instrumentation CRD

Declares auto-instrumentation settings such as agents, exporter targets, environment variables, and runtime options.
